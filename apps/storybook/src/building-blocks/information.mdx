# Information text

Kinda like a text block but with an icon before the text and shown on the result page.

  {common.thead}

      </td>
      <td>Yes</td>
    </tr>
    <tr>
      <td>
        <code>id</code>
      </td>
      <td>The id of the information block</td>

    </tr>

        <code>text</code>
